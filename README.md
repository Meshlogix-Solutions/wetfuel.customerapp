# WetFuel Customer Mobile UI

A standalone Ionic 8 + Angular 22 customer application prototype based on the WetFuel SRS.

## Included workflows

- Customer authentication and verification
- Dashboard with active-delivery summary
- Four-step fuel ordering flow
- Order list, status timeline and live tracking UI
- Customer locations and equipment management
- Completed delivery records and delivery proof
- Invoice and payment-history views
- Notifications, account settings and customer support
- Responsive Ionic/Capacitor-ready project structure

## Prerequisites

- Node.js 22.22.3+ (or Node.js 24.15.0+)
- npm 8+

## Run locally

```bash
npm install
npm start
```

Open `http://localhost:4200`.

## Production integrations still required

This package is a UI prototype with mock data. Connect authentication, backend APIs, Google Maps, push notifications, live GPS, camera/QR features, QuickBooks invoice data, and native Capacitor plugins during implementation.
